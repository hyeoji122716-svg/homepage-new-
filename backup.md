# 세션 백업 — 커넥트유 홈페이지

## 프로젝트 개요
- **프레임워크**: Next.js 16 (App Router) + TypeScript + Tailwind CSS + ESLint
- **목적**: 남궁혜지 강사 (커넥트유 대표) 회사 홈페이지
- **업종**: 미디어 콘텐츠 강연 · 교육 운영 · 마케팅 컨설팅

---

## 파일 구조

```
src/
├── app/
│   ├── globals.css       # Tailwind 설정, scroll-behavior: smooth
│   ├── layout.tsx        # 메타데이터 (title, description), lang="ko"
│   └── page.tsx          # 모든 섹션 조합
└── components/
    ├── Nav.tsx            # 고정 상단 네비, 스크롤 시 배경 전환, 모바일 햄버거
    ├── HeroSection.tsx    # 메인 배너, 핵심 수치 3개, CTA 버튼 2개
    ├── AboutSection.tsx   # 프로필, 전문분야 4개, 경력/학력/자격
    ├── AchievementsSection.tsx  # 통계 4개, 강의 이력 카테고리별, 영상 자리
    ├── PortfolioSection.tsx     # 서비스 3종 카드, 클라이언트 목록, 케이스 스터디
    ├── ContactSection.tsx       # 문의 폼 (제출 완료 화면 포함), 연락처
    └── Footer.tsx               # 저작권, 슬로건
```
